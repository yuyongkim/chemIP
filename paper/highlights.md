# Highlights

- ChemIP unifies nine public data sources (MSDS, patents, trade, drugs, chemical classification, literature) into a single-query web platform for chemical safety decision-making.

- A 134.5 GB patent index covering 350M+ records across six jurisdictions (USPTO, EPO, WIPO, JPO, KIPRIS, CNIPA) enables sub-second chemical-to-patent association using Aho-Corasick multi-pattern matching.

- A locally deployed LLM generates evidence-grounded safety syntheses with post-generation hallucination sanitization and automatic rule-based fallback.

- Controlled scenario evaluations across three representative use cases show a 78% reduction in per-substance investigation time.

- Fully open-source (AGPL-3.0) with 19 automated tests, CI/CD pipeline, and single-command installation.
