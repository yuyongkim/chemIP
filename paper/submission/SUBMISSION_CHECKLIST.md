# SoftwareX Submission Checklist

## Manuscript
- [x] paper_en.tex — within 3,000 word limit (2,175 words)
- [x] Code Metadata table (C1–C9)
- [x] Abstract ~100 words, no over-claims
- [x] Keywords ≤ 6
- [x] Required sections: Motivation / Software Description / Illustrative Examples / Impact / Conclusions
- [x] Figures ≤ 6 (4 used, English UI)
- [x] References — all accessible URLs, no future-dated items
- [x] Declaration of Competing Interest
- [x] Acknowledgments

- [x] Highlights (5 bullets, ≤85 chars each)
- [x] English screenshots (screenshots_en/)
- [x] Revision log (REVISION_LOG.md)
