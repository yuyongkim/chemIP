# ChemIP — Target Journal Comparison

## Top 3 Candidates

| | **SoftwareX** | **JOSS** | **J. Cheminformatics** |
|---|---|---|---|
| Publisher | Elsevier | Open Journals (NumFOCUS) | Springer/BMC |
| IF (2024) | 2.7 | N/A (indexed in Scopus) | ~8.0 |
| APC | **$1,560** | **Free** | $2,390 |
| Review time | ~16 weeks | ~4–8 weeks | ~14 weeks |
| Word limit | 3,000 | **1,000** | No limit |
| Paper ready? | **Yes** (current format) | Need to cut to 1/2 | Need to expand 2–3x |
| Reviewer tests SW? | Sometimes | **Always** (install+run) | Sometimes |
| Independent OK? | Yes | Yes | Yes |
| Acceptance est. | 50–60% | 45–60% | 40–55% |

## Recommendation

### 1st choice: SoftwareX
- Paper is already formatted and within word limit
- Minimal revision needed
- Reviewers may or may not run the software
- APC is moderate ($1,560)

### 2nd choice: JOSS (Journal of Open Source Software)
- **Free** — no publication cost
- Fast review, open process on GitHub
- Reviewers **must** install and run the software — the 134.5GB patent DB is a barrier
- Need to provide a "lite mode" or hosted demo for review
- Paper must be cut to ~1,000 words (basically a summary)

### 3rd choice: Journal of Cheminformatics
- Highest IF (~8.0), best for CV
- Has explicit "Software" article type — good fit
- But: APC is $2,390, paper needs expansion (user study, deeper related work, 30+ references)
- Worth targeting after SoftwareX acceptance for a follow-up expanded paper

## Strategy
1. Submit to **SoftwareX** now (paper is ready)
2. If rejected, cut to 1,000 words → submit to **JOSS** (free, fast)
3. After publication, write expanded methods paper → **J. Cheminformatics**
