# Cover Letter

Dear Editor-in-Chief,

I am pleased to submit the manuscript entitled "ChemIP: An Open-Source Web Platform for Unified Chemical Safety Data Retrieval across Korean and International Regulatory Sources" for consideration as an Article in ACS Chemical Health & Safety.

This manuscript presents ChemIP, an open-source web platform that integrates Korean and international safety, regulatory, and patent data sources into a single-query workflow for chemical safety practitioners. In the Republic of Korea, evaluating a single substance for regulatory compliance currently requires accessing multiple disconnected government portals — KOSHA for MSDS data, KISCHEM and NCIS for substance classifications, KIPRIS for patent status, and MFDS for pharmaceutical registrations — each with different interfaces, authentication, and query formats. ChemIP consolidates these sources so that a practitioner can retrieve MSDS data with GHS classifications, Korean regulatory classifications, patent landscape results, pharmaceutical cross-references, and an AI-assisted safety summary from a single search. A preliminary workflow analysis suggests a potential time reduction from approximately 150 min to 35 min per substance evaluation.

I believe this work is relevant to ACS Chemical Health & Safety for the following reasons:

1. **Chemical safety focus.** The platform's primary purpose is to streamline MSDS retrieval, GHS classification review, and regulatory compliance checking — core topics within the journal's scope. The integration of patent landscape search within the safety evaluation workflow enables practitioners to assess both compliance requirements and freedom-to-operate constraints simultaneously.

2. **Practical tool for safety practitioners.** Recent publications in ACS CHS, such as the IHSCE laboratory safety management tool, demonstrate the journal's interest in software tools that improve chemical safety workflows. ChemIP addresses a comparable need for chemical substance evaluation in the Korean regulatory context.

3. **Open-source and reproducible.** The complete source code is publicly available under the AGPL-3.0 license (https://github.com/yuyongkim/chemIP), with an archived snapshot on Zenodo (DOI: 10.5281/zenodo.19447588), automated tests, and CI/CD configuration. A step-by-step installation guide is provided in the repository README.
