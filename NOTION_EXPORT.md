# 🧪 ChemIP-Platform: 화학물질 안전 & 지식재산권 통합 플랫폼

> Notion 복사 붙여넣기용 정리 문서입니다.

---

## 📌 1. 프로젝트 개요 (Overview)

화학물질 데이터(MSDS)와 글로벌 특허 정보(Patent)를 통합하여, 연구개발 및 안전 관리 효율성을 극대화하는 **올인원 화학 지식 플랫폼**입니다.

- **핵심 가치**: 별도로 존재하는 MSDS 안전 정보와 특허 기술 정보를 하나의 인터페이스에서 즉시 연결.
- **기술적 특징**: 대용량 특허 데이터(140GB+)의 로컬 인덱싱을 통한 초고속 검색 및 Aho-Corasick 알고리즘 기반 키워드 매칭.

## 🏆 3. 주요 성과 (Achievements)

### 🚀 **기술적 성과**

1. **대용량 데이터 처리 파이프라인 구축**
    - 1.6TB 규모의 원본 특허 데이터를 **140GB 고성능 SQLite 인덱스**로 압축 및 최적화 성공.
    - 처리 속도: Aho-Corasick 알고리즘 적용으로 다중 패턴 매칭 속도 **500% 이상 향상** (순차 처리 대비).
2. **하이브리드 시스템 아키텍처 완성**
    - **Frontend**: Next.js 16 + TypeScript + Tailwind CSS로 모던하고 반응주도적인 UI 구현.
    - **Backend**: FastAPI + Python Multiprocessing으로 고의존성 연산 처리 최적화.

### 📊 **데이터 자산 확보**

- **MSDS**: 16개 섹션(유해성, 응급조치 등) 완전 정형화.
- **Patent**: USPTO(미국), EPO(유럽), WIPO(국제), JPO(일본), KIPRIS(한국), CNIPA(중국) 6대 특허청 데이터 통합.

## 💰 7. 수익 모델 (Revenue Models)

앱 기능을 단계별로 유료화하는 **Tiered Freemium** 전략이 가장 적합합니다.

### 💼 B2B 구독 모델 (Subscription)

- **Target**: 기업 연구소, 대학, 화학 제조사
- **Pro Plan (월 $50/user)**: 무제한 특허 검색, 법적 규제 자동 알림, API 접근 권한 제공.
- **Enterprise Plan (별도 협의)**: 사내 ERP/LIMS 시스템 연동, 전용 구축형 서버 제공, 커스텀 데이터 인덱싱.

### 🔓 부분 유료화 (Freemium)

- **Free**: 기본 MSDS 조회, 단순 키워드 검색 (일일 횟수 제한).
- **In-App Purchase**:
  - **심층 리포트 구매 (건당 $5)**: 특정 화학물질에 대한 글로벌 특허 분석 리포트(PDF/Excel) 생성.
  - **전문가 상담 연결**: 검색 결과와 관련된 변리사 또는 노무사 매칭 수수료.

### 🏭 데이터 라이선싱 (Data Licensing)

- **API Sales**: 제약사, AI 신약 개발 스타트업 등에 정제된 화학-특허 매핑 데이터셋 판매.
- **DaaS (Data as a Service)**: 실시간 업데이트되는 규제 DB 접근권 판매.

### 📣 타겟 광고 (Niche Advertising)

- 화학물질 검색 결과 페이지에 관련 **실험 장비(시약, 초자), 안전 보호구(PPE)** 쇼핑몰 광고 노출.
- 전문성이 강한 버티컬 플랫폼이므로 일반 광고 대비 **높은 단가(CPM/CPC)** 책정 가능.
