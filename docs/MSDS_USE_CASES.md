# MSDS 활용 사례 (제출용)

## 1. 프로젝트 개요
ChemIP Platform은 MSDS 중심의 안전보건 정보를 기반으로 특허(IP), 무역/시장, 의약품 데이터를 통합 조회하여 실무 의사결정을 지원하는 시스템입니다.

## 2. 활용 시나리오

### 시나리오 A: 화학물질 도입 전 안전성 검토
- 목적: 신규 원료 도입 전 현장 안전 리스크 식별
- 절차:
  1. 물질명/CAS No. 검색
  2. MSDS 16개 섹션 확인(유해성, 응급조치, 취급/보관, 노출방지)
  3. 작업 표준서(SOP) 및 교육 항목 반영
- 산출물:
  - 핵심 위험요인 요약
  - 필수 PPE 체크리스트
  - 비상대응 포인트

### 시나리오 B: 수출 준비를 위한 통합 검토
- 목적: 해외 진출 전 규제 및 시장/특허 리스크 동시 점검
- 절차:
  1. MSDS 상 위험분류/규제 관련 항목 확인
  2. 국가/시장 정보 API로 진입 환경 조사
  3. 연관 특허 검색으로 분쟁 가능성 사전 검토
- 산출물:
  - 국가별 주의사항
  - 시장 진입 참고 정보
  - 특허 리스크 후보 목록

### 시나리오 C: 의약품 성분 교차 조사
- 목적: 의약품 성분 관련 자료를 단일 화면에서 확인
- 절차:
  1. 의약품명 검색
  2. 허가정보 + e약은요 확인
  3. 동일 키워드 기반 특허/시장 정보 교차 조회
- 산출물:
  - 제품별 요약 카드
  - 성분 중심 연관 정보 묶음

## 3. 기대 효과
- 데이터 탐색 시간 단축: 분산된 공공 데이터를 단일 경로로 조회
- 안전성 검토 일관성 확보: MSDS 기반 표준 점검 흐름 유지
- 사전 리스크 관리: 시장/특허 이슈를 초기 단계에서 식별

## 4. 시연 절차 (요약)
1. 홈에서 화학물질 검색
2. 상세 화면에서 MSDS 섹션 확인
3. 특허/시장 탭으로 이동해 연관 정보 확인
4. 필요 시 의약품 탭에서 동일 키워드 재검증
