# ChemIP 파일럿 KPI 측정 템플릿

## 목적
- 도입 효과를 "체감"이 아니라 "수치"로 검증한다.
- 최소 4주 동안 같은 기준으로 측정해 부서 확산 여부를 판단한다.

## 측정 원칙
- 비교 기준은 `기존 방식 vs ChemIP 사용 방식`으로 통일
- 동일 업무 유형(예: 물질 사전검토, 규제 검토, 특허 스크리닝)끼리 비교
- 1주 단위 집계 + 월말 종합

## KPI 정의
| KPI | 정의 | 목표(예시) | 수집 위치 |
|---|---|---|---|
| 검색~1차판단 시간 | 검색 시작부터 내부 1차 의사결정까지 소요 시간(분) | 40% 단축 | 사용자 입력 시트 |
| 건당 재작업 횟수 | 규제/특허 누락으로 인한 재검토 횟수 | 30% 감소 | 이슈 트래커 |
| API 호출량 | 외부 API 총 호출 수(주간) | 기준 대비 최적화 | 서버 로그 |
| 화면 응답시간 | 주요 페이지 P95 응답시간(ms) | 2초 이내 | 모니터링 로그 |
| 사용자 만족도 | 5점 척도 설문 | 4.2/5 이상 | 주간 설문 |

## 월말 의사결정 기준 (Go / Hold / Expand)
- `Go`: 핵심 KPI 3개 이상 목표 달성 + 장애 중대 이슈 없음
- `Hold`: KPI 개선은 있으나 품질/안정성 보완 필요
- `Expand`: 2개 이상 부서에서 재사용 의사 확인 + KPI 안정 달성

## 권장 운영 루틴
1. 월요일: 대상 업무/측정 범위 확정
2. 수요일: 중간 점검(데이터 누락 확인)
3. 금요일: 주간 집계 + 15분 리뷰
4. 월말: 4주 누적 분석 및 확산 여부 결정
